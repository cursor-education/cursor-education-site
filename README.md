## Setup

> ### on local
>
> ```bash
> $ boot2docker up
> $ make PORT=1234
> $ open http://{site-address}:1234/
> ```
> 
> ### on stage:
> ```bash
> $ curl -sSL https://get.docker.com/ | sh
> $ docker -v
> $ sudo service docker restart
> $ git clone https://github.com/itspoma/cursor-education-site site/
> $ cd site/ && make PORT=1234
> $ open http://{site-address}:1234/
> ```
